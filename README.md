# json-tools

一个处理 JSON 的的在线工具

[![json-tools](https://www.gausszhou.top/static/data/i/github/json-tools.webp)](https://gausszhou.github.io/json-tools/)

## Features

- [x] JSON 压缩
- [x] JSON 格式化
- [x] JSON 排序
- [ ] JSON 深度合并
- [x] JSON 深度解析
- [ ] JSON 转 TypeScript
- [ ] JSON 嵌套转扁平
- [ ] JSON 扁平转嵌套
- [ ] JSON 转 Object
- [ ] Object 转 JSON
- [ ] JSON 转 YAML
- [ ] YAML 转 JSON
