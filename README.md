# json-tools

一个处理 JSON 的的在线工具

[![json-tools](https://www.gausszhou.top/static/data/github/json-tools/1.png)](https://gausszhou.github.io/json-tools/)

## Features

- [x] JSON 压缩
- [x] JSON 格式化
- [x] JSON 排序
- [ ] JSON 深度合并
- [x] JSON 深度解析
- [x] JSON 转 TypeScript
- [x] JSON 嵌套转扁平
- [x] JSON 扁平转嵌套
- [ ] JSON 转 Object
- [ ] Object 转 JSON
- [x] JSON 转 YAML
- [x] YAML 转 JSON
- [x] CSV 转 JSON
- [x] JSON 转 CSV
