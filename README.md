# json-tools

强大的 JSON 处理工具集

[![json-tools](https://www.gausszhou.top/static/data/github/json-tools/2.png)](https://gausstool.github.io/json-tools/)

## Features

### 核心工具

- [x] JSON 压缩
- [x] JSON 格式化
- [x] JSON 排序
- [x] JSON 深度解析
- [x] JSON 嵌套转扁平
- [x] JSON 扁平转嵌套
- [x] JSON 转接口代码

### 格式转换

- [x] Object 转 JSON
- [x] JSON 转 Object
- [x] JSON 转 YAML
- [x] YAML 转 JSON
- [x] JSON 转 CSV
- [x] CSV 转 JSON
